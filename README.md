# Neuro Sourced — B2B Wholesale Peptide Command Center

## Quick Start

```bash
npm install
npm run dev
```

Opens at http://localhost:3000

## Setup

1. Clone or unzip this project
2. Run `npm install`
3. (Optional) Add your Anthropic API key to `.env` for AI workers
4. Run `npm run dev`

## Features

- **HQ Dashboard** — KPIs, AI workers, live activity feed
- **Quick Generate** — Instant lead generation (+25, +50, +100, +200)
- **Lead Management** — Full CRM with search, filters, smart lists
- **Pipeline** — Kanban board across 6 stages
- **Email Drafts** — Auto-generated with real product catalog per vertical
- **Call Tracking** — Log outcomes, track follow-ups
- **Walk-In Tracker** — Friday/Saturday field day logging
- **Daily Checklist** — Smart task list based on pipeline state
- **Dispatch Sync** — Batch export for Claude Dispatch → Gmail
- **Product Catalog** — Full Neuro Labs wholesale catalog (45+ peptides)

## AI Workers

- **Lead Scout** — Finds leads via web search (needs API key)
- **Email Ops** — Drafts personalized emails (runs locally, no API needed)
- **Organizer** — Auto-manages pipeline stages (runs locally)
- **Call Prep** — Researches leads before calls (needs API key)
